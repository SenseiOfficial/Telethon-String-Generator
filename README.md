# TG String Session
Generate Telethon String Session Using this bot.




## Configs:
- API_HASH
  - Get from [Here](https://my.telegram.org).
- API_ID
  - Get from [Here](https://my.telegram.org).
- BOT_TOKEN
  - Telegram Bot Token from [here](https://telegram.dog/BotFather).
